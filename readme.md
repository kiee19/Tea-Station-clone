# Tea Station Clone

Dự án landing page **Tea Station Clone** được xây dựng với các công nghệ sau:

- **HTML**: Xây dựng cấu trúc và nội dung trang web.
- **CSS**: Tùy biến giao diện và bố cục, kết hợp với Tailwind CSS.
- **Tailwind CSS**: Utility-first CSS framework dùng để thiết kế UI nhanh chóng và nhất quán.
- **JavaScript**: Thêm tương tác và xử lý logic trên giao diện (frontend).
- **Node.js & npm**: Quản lý package và script phục vụ quá trình phát triển (build CSS, chạy dev server, v.v.).
